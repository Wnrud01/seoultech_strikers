# 🍐 로컬몬 · 전남

전라남도 특산물을 모티브로 한 캐릭터 **로컬몬**을 수집하며 지역 전통시장을 탐험하는 위치기반 AI 서비스의 프론트엔드 프로토타입입니다.

> AI+X 산학연계 해커톤 · Team **Strikers**

## 서비스 소개

전남에는 나주 배, 영광 굴비, 완도 전복 등 전국적으로 유명한 특산물이 많지만, 정작 관광객의 실구매로 이어지지 못하고 지역 상권은 약화되고 있습니다. **로컬몬 · 전남**은 이 문제를 "방문 → 구매 → 재방문"으로 이어지는 게임형 선순환 구조로 풀어냅니다.

- 지도에서 오늘의 **핫스팟 지역**을 확인하고 시장을 방문
- 구매 영수증을 **OCR로 촬영·인증**
- 전남 특산물이면 일반 상품 대비 **10배 경험치**의 특산물 먹이 획득
- 먹이를 먹여 로컬몬을 성장시키고, 랭크업 시 **지역화폐 포인트** 지급

## 핵심 화면 흐름

| 탭 | 내용 |
|---|---|
| 🏠 홈 | 보유 포인트, 오늘의 핫스팟 지도, 일일 미션 |
| 🏆 업적 | 일일 미션 + 특산물 마니아 · 전남 순례자 등 업적 리스트 |
| 🐾 몬스터 | 수집한 로컬몬 도감 (지역별 8종) |
| 👤 내정보 | 포인트 현황, 지역화폐 전환 |
| 🧾 영수증 인증 | 상단바 버튼으로 언제든 진입, OCR 인식 → 먹이 획득 |

로컬몬을 탭하면 하단 시트로 **경험치 바 · 제철 정보 · AI 도슨트(한국어/English) · 먹이 주기** 상세 화면이 열립니다.

## 로컬몬 진화 시스템 — 배돌이(나주 배) 예시

경험치를 채워 랭크업할 때마다 외형이 변화합니다. 아래는 나주 배 캐릭터 **배돌이**의 5단계 진화 예시입니다.

| ★★★★★ 전설 | 신성한 배령돌이 | 30,000 | `characters/baedori_5.png` |

## 폴더 구조 (권장)

```
.
├── index.html          # 앱 전체 (단일 파일, 프론트엔드)
├── characters/         # 로컬몬 이미지 (baedori_1~5.png, nakjiwang.png, nokcha.png 등)
├── img/
│   └── jeonnam.jpg     # 홈 화면 전남 지도 이미지
└── api/                # (예정) OCR 연동 백엔드
```

## 기술 스택

- **프론트엔드**: Vanilla HTML/CSS/JS 단일 파일 (프레임워크 없음, 토스 스타일 UI)
- **OCR**: `index.html`이 `POST /api/ocr`로 영수증 이미지를 전송하도록 구현되어 있으며, 백엔드가 아직 없을 경우 자동으로 데모용 샘플 텍스트로 대체 동작합니다. 실제 연동 시 네이버 CLOVA OCR 또는 Google Vision API 응답에서 인식 텍스트만 뽑아 그대로 반환하면 됩니다.
- **데이터**: `REGIONS`, `DAILY`, `ACHIEVEMENTS` 등 상단 DATA 블록만 수정하면 지역·캐릭터·미션 내용을 바꿀 수 있습니다.

## 실행 방법

```bash
# 별도 설치 없이 브라우저에서 바로 열기
open index.html

# 또는 로컬 서버로 실행 (상대 경로 이미지 로딩 안정성 위해 권장)
npx serve .
```
